## Installing the Chart

To install the chart with the release name `hugs`:

```console
$ helm repo add hugs https://github.com/huggingface/hugs-helm-chart/releases/download/0.0.1/hugs-0.0.1.tgz
$ helm repo update
$ helm install hugs hugs/hugs
```
